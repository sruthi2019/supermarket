This is a simple Python-based Super Market Billing System.  
It allows users to select items, enter quantities, and generate a bill with GST.

## Features
- Displays available items with prices  
- Accepts multiple item purchases  
- Calculates total amount  
- Adds 5% GST  
- Generates a formatted bill with date and customer name  

## Technologies Used
- Python
- datetime module

## How to Run
1. Install Python on your system  
2. Download or clone this repository  
3. Open terminal / command prompt  
4. Run the program:

   python super_market.py
